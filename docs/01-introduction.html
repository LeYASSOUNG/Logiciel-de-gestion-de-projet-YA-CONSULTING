01
Présentation du projet
Contexte, objectifs et fonctionnalités principales

Le Logiciel de Gestion de Projet YA CONSULTING est une application web interne de gestion financière et opérationnelle. Elle centralise le suivi des projets, dépenses et la production de rapports financiers mensuels.

Fonctionnalités principales

Module Description
🗂️ Projets Création, édition, suivi de statut (en cours / terminé / en pause), gestion des budgets en FCFA
💰 Dépenses Saisie catégorisée avec justificatifs (PDF, images), historique complet
📊 Rentabilité Calcul automatique : gain brut = budget − dépenses, taux de rentabilité en %
📄 Rapports Génération de rapports mensuels PDF (DomPDF) et CSV avec comparaison N vs N-1
👥 Utilisateurs Gestion des accès par rôles (Administrateur, Chef de projet)
📋 Audit Journal d'activité complet (Spatie Activitylog) pour toutes les actions critiques

03
Architecture générale
Structure du projet et flux de données

Le projet suit l'architecture MVC Laravel couplée au paradigme Inertia.js : les contrôleurs PHP passent les données directement comme props Vue.js, éliminant le besoin d'une API REST séparée.
