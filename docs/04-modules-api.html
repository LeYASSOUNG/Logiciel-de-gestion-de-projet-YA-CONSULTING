07
Modèles Eloquent
Attributs calculés, relations et scopes

Project — Attributs calculés (accesseurs)

Attribut Calcul Type
total_expenses SUM(expenses.amount) via Eloquent float
gross_gain budget − total_expenses float
profitability_rate (gross_gain / budget) × 100 arrondi à 2 décimales float (%)
is_profitable gross_gain >= 0 bool

Project — Scopes de requête

Scope SQL équivalent
scopeEnCours($q) WHERE status = 'en_cours'
scopeTermine($q) WHERE status = 'termine'
scopeEnPause($q) WHERE status = 'en_pause'
scopeByClient($q, $id) WHERE client_id = $id
scopeByYear($q, $year) WHERE YEAR(start_date) = $year
08
Contrôleurs
Actions, validations et données retournées

ProjectController — Règles de validation

Champ Règles
name required | string | max:255
client_id required | exists:clients,id
start_date required | date
planned_end_date required | date | after_or_equal:start_date
actual_end_date nullable | date | after_or_equal:start_date (update uniquement)
budget required | numeric | min:0
status required | in:en_cours,termine,en_pause
supplier_contact nullable | string | max:255

ExpenseController — Règles de validation

Champ Règles
project_id required | exists:projects,id
category_id required | exists:expense_categories,id
date required | date
amount required | numeric | min:0.01
receipt nullable | file | max:5120 | mimes:pdf,jpeg,png,jpg,webp
