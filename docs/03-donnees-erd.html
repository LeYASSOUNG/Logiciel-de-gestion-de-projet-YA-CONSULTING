05
Guide d'installation
Mise en place de l'environnement de développement

Prérequis

Outil Version minimale
PHP ≥ 8.2
Composer ≥ 2.x
Node.js ≥ 18.x
npm ≥ 9.x

Étapes d'installation

# 1. Cloner le dépôt
git clone <url-du-depot>
cd ya-consulting-app

# 2. Dépendances PHP
composer install

# 3. Dépendances JavaScript
npm install

# 4. Configurer l'environnement
cp .env.example .env
php artisan key:generate

# 5. Base de données SQLite
touch database/database.sqlite
php artisan migrate --seed

# 6. Lien symbolique pour les fichiers publics
php artisan storage:link

# 7. Lancer le serveur de développement
composer run dev
ℹ️ Commande composer run dev Lance simultanément : serveur Laravel (port 8000) + Vite hot-reload + queue:listen + pail (logs). Accès : http://localhost:8000
